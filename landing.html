Structural Design Patterns

Seminar presentation and Python implementation demonstrating Structural Design Patterns.

Summary

This artefact builds upon the Structural Design Pattern concepts introduced in Unit 4. A coffee ordering system was developed using the Decorator and Composite patterns to demonstrate how functionality can be extended without modifying existing code and how individual objects can be composed into larger structures.

Learning & Reflection

Key Learning

  • Decorator Pattern
  • Composite Pattern
  • Structural Design Patterns
  • Object Composition
  • Extensible Software Design

Developing the coffee ordering system improved my understanding of how structural design patterns can increase flexibility and maintainability. The activity demonstrated how behaviour can be extended through decoration and how complex object hierarchies can be represented through composition.
